# entre-les-pages
Une application web permettant de découvrir, rechercher et gérer une collection de livres en toute simplicité.

## À propos

**Entre les Pages** est une application web moderne conçue pour offrir une expérience fluide de découverte, de recherche et de gestion d'une collection de livres. Les utilisateurs peuvent explorer un large catalogue, rechercher des ouvrages par titre ou par auteur, consulter leurs détails et organiser leurs livres préférés grâce à une interface intuitive.

Ce projet a été réalisé dans le but de mettre en pratique le développement web full-stack tout en privilégiant une architecture propre, un design responsive et une expérience utilisateur agréable.

---

## Fonctionnalités
- Parcourir une collection de livres
- Rechercher des livres par titre ou par auteur
- Consulter les détails d'un livre
- Gérer une liste de livres favoris
- Interface utilisateur responsive
